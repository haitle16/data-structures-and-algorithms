## Stacks and Queues

- For Stack
- - Implemented the push functionality so user can push any value (except null) onto the top of the stack
- - Implemented the pop functionality for users to have the ability to remove value off the top of the stack.
- - Implemented the peek functionality for user to see the value of the stack on top.

- For Queues
- - User will be able to enqueue into the front of the queue.
- - User can use dequeue and remove the first value that was stored in the queue
